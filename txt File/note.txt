Operation	Meaning
-------------------------
Create		Add new data (e.g., save a new user or product)
Read		Get data (e.g., view all products, get a user by ID)
Update		Modify existing data (e.g., change the price of a product)
Delete		Remove data (e.g., delete a user from the database)


| Method     | Purpose     | Action                         |
| ---------- | ----------- | ------------------------------ |
| **GET**    | Read data   | Get product info, user list    |
| **POST**   | Create data | Add new product, register user |
| **PUT**    | Update data | Edit product/user info         |
| **DELETE** | Delete data | Remove product/user            |
